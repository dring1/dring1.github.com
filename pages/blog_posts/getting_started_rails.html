One of the most daunting tasks for getting started with Rails may be the fact that installing it can be quite a bitch. So, for your linux users I have supplied a quick bash script to simplify the task.

What installs?
  • RVM
  • Ruby 1.9.3
  • Rails 3
  • nodejs
  • A smack load of libraries
  • cURL
Additionally the script will attempt to load the rvm environment on login. If you wish to use more modern versions of software you must edit the script yourself. You must run the script from a login shell inorder to install correctly. The script will only function on Linux with bash.
